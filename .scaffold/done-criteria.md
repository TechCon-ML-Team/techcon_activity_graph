# Критерии готовности: github-readme-activity-graph

> Агент проверяет этот список перед тем как объявить задачу выполненной.

## Обязательные критерии (всегда)

- [ ] Все изменения закоммичены (`git status` чистый)
- [ ] Коммит в формате Conventional Commits
- [ ] Изменения существуют в репозитории, не только локально/на сервере
- [ ] Тесты запущены (Jest) и зелёные
- [ ] Новый код покрыт тестами (или задокументировано почему нет)

## Persistence contract

- [ ] Конфигурационные изменения в репо (IaC, не вручную на сервере)
- [ ] Секреты не закоммичены (только через env vars / secrets manager)
- [ ] CI (GitHub Actions) не сломан

## Что НЕ считается выполненным

- Файл создан но не закоммичен
- Работает локально но нет тестов
- Конфиг изменён вручную на сервере (не в IaC)
- Есть TODO/FIXME оставленные в коде без тикета
